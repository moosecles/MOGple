# MOG-PLE
**'Maple Optimizations Generator'** — an unofficial fan-made planning tool for MapleStory Classic (CBT v0.26–0.28, Victoria Island).

## What it does

- **Character Builder** — pick your class, allocate stats, and equip gear to see your damage range in real time
- **Training Advisor** — get recommended grinding spots based on your current level and stats
- **Item Progression** — see the best gear upgrades for each equipment slot, including future tier items and their damage impact

## Stack

Vite + React 18 + TypeScript + Tailwind CSS, deployed to GitHub Pages.
